Annotations Help

To add an annotation to the scene, you can either place it or type in the specific coordinates. You can place it by clicking on the button then setting the marker on the scene.
The marker is draggable after being set. If you choose to type in your own coordinates, click the same button to preview it on the scene.


Once you're happy with its position, name your annotation, add a short description and confirm it using the button The description can be text, but it can also be an image, a video, etc.
